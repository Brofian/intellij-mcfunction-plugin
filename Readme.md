# IntelliJ McFunction Support

An IntelliJ Plugin to implement Syntax Highlighting and Syntax Error checks for the Minecraft-Datapack 
filetype `.mcfunction`.

*Currently Supported Minecraft Version:* **1.19.4**


- [x] Syntax Highlighting
- [x] Syntax error detection
- [ ] Code completion
- [ ] Context aware code completion
- [ ] Plugin published via Jetbrains Marketplace

## Important
This is a hobby project and I do this in my free time. So there won't be regularly updates. But if you encounter
a problem or a bug, feel free to open an issue or even a pull request on the GitHub repository.
